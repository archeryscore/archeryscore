## 📖 Cos'è ArcheryScore?

ArcheryScore è una web application dedicata agli arcieri che permette di registrare gare e allenamenti, monitorare i propri progressi e analizzare statistiche avanzate.

Nasce con un obiettivo semplice:

> trasformare un semplice archivio di punteggi in un vero diario digitale dell'arciere.

# 📊 Analisi disponibili

ArcheryScore analizza automaticamente:

- media generale
- media ultime 5 gare
- migliore prestazione
- peggior prestazione
- record personali
- confronto con la media
- andamento mensile
- distribuzione dei valori
- numero di frecce
- media per freccia
- classifica personale

separando completamente:

🏹 Gare

🎯 Allenamenti

e ogni tipologia di disciplina.

---

# 🏅 Sistema Ricompense

L'app assegna automaticamente badge e medaglie in base ai risultati ottenuti.

Ogni ricompensa viene salvata con:

- nome
- descrizione
- data di ottenimento

---

# 🗺️ Mappa

Visualizzazione delle competizioni disputate con:

- località
- province
- regioni

e storico consultabile.
